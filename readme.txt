Для запуска проекта необходим Maven (в случае отсутствия- напишите на почту - соберу без него)

Лог файл - MyAppLog.log
Настройки в папке /property
Javadoc в папке /doc

Выгрузка БД - DepDB.sql (работаю с БД через phpMyAdmin импортировал из него)
!!Важно! в db.properties указан НЕ 3306 (mysql порт)
Программа работает в цикле while до ввода колючего слова exit
Создание xml вне папки проекта не предусмотрено, как и синхронизация

Запуск через bat файл (далее работа с консолью)
